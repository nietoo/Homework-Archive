
Правительство России выполнило до 15 июля, то есть в срок, поставленную в указе президента Владимира Путина задачу по созданию в стране широкой системы раскрытия информации госорганов в формате открытых данных.

"Вместо 500 пакетов данных, раскрытие которых планировалось, раскрыто более 1 тыс. пакетов. 45 министерств и ведомств создали на своих сайтах страницы с открытыми данными", - сообщил министр РФ по связям с Открытым правительством Михаил Абызов в понедельник на расширенном заседании Совета по открытым данным.

В числе реализованных крупных проектов он назвал единый портал бюджетной системы, портал криминальной статистики, единый портал госзакупок. 

По словам заместителя министра экономического развития Сергея Белякова, из российских регионов наиболее далеко продвинулись Москва (198 наборов данных), Амурская (43 набора данных) и Ульяновская (25) области, а также Краснодар и Южно-Сахалинск. У граждан и бизнеса наибольшим спросом пользуются наборы данных, связанных с образованием, ЖКХ, здравоохранением, статистической информацией, сказал он.

По словам Абызова, конечной задачей создания нового механизма раскрытия информации, который, в частности, дает возможность создавать востребованные гражданами и обществом интернет-сервисы, является повышение качества жизни людей и качества среды их обитания.

Участники совещания назвали также проблемы, мешающие реализации проекта. Так, заместитель министра связи и массовых коммуникаций Денис Свердлов указал на несогласованность баз данных ведомств.

"У нас ведомственная информатизация проходила децентрализованно, - отметил он. - Поэтому данные, справочники не синхронизированы. Необходимо определить наиболее востребованные типы открытых данных и создать для них общие регламенты актуализации, обработки".

Другой проблемой Свердлов назвал отсутствие экосистемы разработчиков приложений на основе открытых данных. По его мнению, для решения этой проблемы необходимо создать единую площадку взаимодействия разработчиков с госорганами, где они могли бы получать техподдержку, API для разработки приложений и т.п.

И.о. министра правительства Москвы Артем Ермолаев выделил также проблему актуализации информации. По его мнению, необходимо выработать механизмы оперативной актуализации наборов открытых данных с участием самих граждан, которые указывают на те или иные ошибки в базах данных.

По оценке директора некоммерческого партнерства "Информационная культура" Ивана Бегтина, существует проблема с качеством страниц открытых данных. По его словам, нередки случаи, когда за ссылкой на конкретный набор открытых данных нет соответствующего файла, или наборы представлены только ссылкой - не имеют аннотации или паспорта файла, из которых можно было бы понять, какая информация содержится в конкретном пакете.

Говоря о перспективах проекта, Абызов заявил, что во втором полугодии 2013 года планируется раскрыть еще 500 востребованных и качественных датасетов, при этом 10 пакетов данных впервые должны быть предоставлены в качестве "пять звезд" (наборы данных, которые можно автоматически связать между собой). Еще одной задачей является создание востребованных приложений на основе открытых данных, отметил он.

Кроме того, будет вестись совершенствование нормативной базы, обучение чиновников работе с открытыми данными, разработка Центрального портала открытых данных.

Россия, заявил Абызов, будет работать над реализацией принципов Хартии открытых данных, принятой на саммите G8 в июне этого года. 

Сейчас Россия занимает не лучшее место среди стран мира по открытым данным - 25-е из 120 государств и 8-е среди стран "восьмерки". Задачей-минимум участники заседания назвали вхождение России по этому параметру в пятерку в G8. Для достижения этой цели планируется, в частности, разработать дорожную карту внедрения Хартии открытых данных.

Абызов считает, что пока России рано соревноваться с лидерами списка - США и Великобританией. "Пробиться в тройку лидеров очень тяжело, - считает Абызов. - Лидеры уже соревнуются не по раскрытию наборов открытых данных, а по экономической эффективности - по созданию новых рабочих мест, по экономии для бюджета страны".

Впрочем, по мнению Абызова, есть три перспективных направления в области открытых данных, движение по которым может позволить России выйти вперед. Называть их министр не стал, отметив только, что Совет по открытым данным будет рассматривать возможность работы по этим направлениям на своем очередном заседании через две недели.

Открытые данные - информация, размещаемая в интернете в формате, который позволяет вести ее автоматизированную обработку без предварительных изменений человеком в целях повторного использования (такой формат называют еще "машиночитаемым"). Открытые данные, как показал опыт США, Великобритании и других стран, позволяют разработчикам создавать новые удобные сервисы в интернете, государству - поддерживать развитие IT-бизнеса и совершенствовать инструменты общественного контроля.

В России координирует эту работу Совет по открытым данным, возглавляемый министром РФ Абызовым.
