
Компания Microsoft вернет кнопку "Пуск" в свою новейшую операционную систему Windows 8. Как сообщает издание The Verge, ключевой элемент предыдущих версий Windows войдет в пакет обновления Windows 8.1, публичное бета-тестирование которого начнется в конце июня. По данным The Verge, отказаться от кнопки после установки Windows 8.1 не получится.

Кнопка "Пуск" присутствовала во всех ОС Microsoft, начиная с Windows 95. В выпущенной в прошлом году Windows 8 этот элемент был заменен на стартовый экран. Это решение Microsoft было болезненно воспринято большим количеством пользователей десктопных систем. Вскоре после выпуска Windows 8 в сторонние компании выпустили множество расширений, копирующих функционал "Пуска".

Windows 8.1, первый большой пакет обновлений для Windows 8, также позволит пользователям загружать систему в режиме рабочего стола, минуя стартовый экран нового интерфейса Metro.
