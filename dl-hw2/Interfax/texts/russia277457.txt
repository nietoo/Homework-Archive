
Работы по созданию инфраструктуры для вертолетоносцев типа "Мистраль" во Владивостоке начнутся в следующем году, сообщили в субботу журналистам в Главном штабе ВМФ.

"Проведение работ по созданию инфраструктуры для двух вертолетоносцев типа "Мистраль", которые должны поступить для базирования во Владивостоке, Спецстрой России планирует начать в 2013 году с расчетом завершения работ по инфраструктуре для первого корабля в четвертом квартале 2014 года", - сказал собеседник агентства.

По его словам, рекогносцировочные работы в районе создания объектов уже завершены.
